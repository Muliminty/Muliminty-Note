## 📖 使用说明

### 目录结构说明

- **MOC 文件**：`!MOC-分类.md` - 各分类的索引文件，包含完整的知识体系
- **README 文件**：各目录下的 README.md - 该目录的说明和学习路径
- **普通文件**：使用中划线分隔，如 `变量声明.md`、`作用域和闭包.md`

### 学习建议

1. **初学者**：按照「学习路径」从基础入门开始，循序渐进
2. **有经验者**：使用「技能分类」快速定位需要补强的领域
3. **查阅参考**：使用各 MOC 文件快速查找特定主题

### 文件命名规范

- **MOC 文件**：`!MOC-分类名.md`（如 `!MOC-前端体系.md`）
- **普通文件**：使用中划线分隔（如 `变量声明.md`）
- **目录名**：使用数字前缀+中划线+名称（如 `01-语言核心`）
