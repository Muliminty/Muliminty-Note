# Muliminty-Note

> 一套正在从“学习路径型知识库”迁移为“个人技术百科”的全栈知识体系，支持**按主题检索**与**按迁移骨架持续整理**。
>
> 覆盖前端、服务端、计算机基础、AI 与机器学习，采用 MOC 索引与 Obsidian 双链，持续维护更新。

---

## 核心特性

- **百科化迁移中**：已建立新的百科一级目录骨架，旧目录按专题逐步迁移
- **MOC 索引体系**：每类知识有 `!MOC-*.md` 索引，便于跳转和复习
- **Obsidian / Quartz 双链**：支持知识图谱、反向链接、全文搜索
- **中文全栈覆盖**：前端、Node/Java、Nginx、Docker、AI 等
