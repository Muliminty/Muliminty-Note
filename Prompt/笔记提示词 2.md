# 🧠 编程学习笔记模板（可直接套用）

---

1）一句话概括主题  
→ 用最简单的话说明是什么

2）它是什么
→ 不讲术语，用比喻、生活化表述  
→ 必须举例帮助理解

3）能解决什么问题 + 为什么重要  
→ 从痛点出发说明价值  
→ 用通俗语言说明重要性

4）核心知识点拆解（分点详细说明）  
- 基本概念（口语化解释）  
- 必会语法/关键关键字 配小例子  
- 关联知识必须使用 **双链**  
  例：变量 → [[数据类型]]  
- 常见误解说明与纠正
- 详细一点 要很深入 有深度的内容 不要流水账

5）示例代码（可运行 + 逐行注释 *至少一个案例*）
```js
// 示例标题
// 每行写清楚在做什么
````

6）常见错误与踩坑（至少三点）

* 错误1 → 为什么错 & 会导致什么
* 错误2 → 为什么容易踩 & 正确方式
* 错误3 → 真实开发场景问题示例

7）实际应用场景
→ 说清楚它能用来做什么
→ 结合真实场景更易理解

8）给新手的练习题（可立即实践）
✔ 基础题
✔ 进阶题
→ 可运行、有明确目标、有输出结果

9）用更简单的话再总结一遍（方便复习）
→ 更短、更口语、更易记忆

10）知识体系延伸 & 继续学习方向 & 遵守仓库规范文档
→ 可学习的下一个知识点
→ 使用原生的 md 格式作为双链的引用方式
