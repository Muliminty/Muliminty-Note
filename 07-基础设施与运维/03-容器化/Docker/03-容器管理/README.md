# Docker 容器管理目录说明

本目录用于收纳 Docker 容器运行时相关的正式知识笔记，覆盖容器生命周期、数据、网络、运行模式与资源限制。

推荐阅读顺序：

1. [01-容器基础操作.md](./01-容器基础操作.md)
2. [04-容器运行模式.md](./04-容器运行模式.md)
3. [02-容器数据管理.md](./02-容器数据管理.md)
4. [03-容器网络基础.md](./03-容器网络基础.md)
5. [05-容器资源限制.md](./05-容器资源限制.md)

本目录优先收录：

- 容器创建、停止、删除与进入容器
- 数据卷、挂载与持久化基础
- 网络模式、端口映射与容器通信
- 运行模式、重启策略与资源限制

本目录不优先收录：

- 镜像构建与 Dockerfile 语法
- 多服务编排
- 生产故障案例的深度排错记录
