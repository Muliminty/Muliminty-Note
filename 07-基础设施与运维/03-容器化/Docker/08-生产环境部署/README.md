# Docker 生产环境部署目录说明

本目录用于收纳 Docker 在生产环境中的部署、监控、编排与持续交付相关知识。

推荐阅读顺序：

1. [01-部署策略.md](./01-部署策略.md)
2. [03-容器编排方案.md](./03-容器编排方案.md)
3. [02-监控与日志.md](./02-监控与日志.md)
4. [04-CI-CD集成.md](./04-CI-CD集成.md)

本目录优先收录：

- 生产部署策略与发布方式
- 容器监控、日志与告警
- Swarm、Kubernetes 等编排方案
- 持续集成与持续部署流程

本目录不优先收录：

- 单机环境基础命令
- Dockerfile 语法细节
- 纯故障复盘记录
