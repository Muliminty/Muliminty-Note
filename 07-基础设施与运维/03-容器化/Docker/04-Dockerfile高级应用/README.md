# Dockerfile 高级应用目录说明

本目录用于收纳 Dockerfile 进阶主题，重点覆盖多阶段构建、参数传递、健康检查与编写最佳实践。

推荐阅读顺序：

1. [02-构建参数与变量.md](./02-构建参数与变量.md)
2. [01-多阶段构建.md](./01-多阶段构建.md)
3. [03-健康检查配置.md](./03-健康检查配置.md)
4. [04-Dockerfile最佳实践.md](./04-Dockerfile最佳实践.md)

本目录优先收录：

- Dockerfile 的高级指令与写法
- 构建优化、参数管理与阶段拆分
- 健康检查、可维护性与安全性实践

本目录不优先收录：

- 镜像基础操作
- 容器运行时管理
- Compose 多服务编排
