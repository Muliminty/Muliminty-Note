# Docker 网络与存储目录说明

本目录用于收纳 Docker 运行时的网络与存储相关知识，重点覆盖数据卷、持久化策略、网络驱动与容器通信。

推荐阅读顺序：

1. [01-存储详解.md](./01-存储详解.md)
2. [02-数据持久化策略.md](./02-数据持久化策略.md)
3. [03-网络详解.md](./03-网络详解.md)

本目录优先收录：

- 存储驱动、数据卷、挂载方式
- 数据备份、恢复、迁移与持久化设计
- 网络驱动、自定义网络与容器通信

本目录不优先收录：

- 单纯的容器生命周期命令
- Compose 文件写法
- 故障排查型临时记录
