# Docker 镜像管理目录说明

本目录用于收纳 Docker 镜像相关的正式知识笔记，涵盖镜像获取、构建、优化、仓库管理与 Dockerfile 编写。

推荐阅读顺序：

1. [03-镜像基础操作.md](./03-镜像基础操作.md)
2. [05-Dockerfile编写指南.md](./05-Dockerfile编写指南.md)
3. [02-镜像构建实践.md](./02-镜像构建实践.md)
4. [04-镜像优化策略.md](./04-镜像优化策略.md)
5. [01-镜像仓库管理.md](./01-镜像仓库管理.md)

本目录优先收录：

- 镜像拉取、标签、导入导出等基础操作
- Dockerfile 编写与镜像构建流程
- 镜像体积与构建性能优化
- 公有与私有仓库的使用和治理

本目录不优先收录：

- 容器运行时管理
- 网络与存储配置
- 生产部署和故障排查
