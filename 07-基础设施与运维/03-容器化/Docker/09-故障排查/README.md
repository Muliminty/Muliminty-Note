# Docker 故障排查目录说明

本目录用于收纳 Docker 在使用与部署过程中常见问题的排查方法、调试技巧与诊断工具说明。

推荐阅读顺序：

1. [01-常见问题排查.md](./01-常见问题排查.md)
2. [02-调试技巧.md](./02-调试技巧.md)
3. [03-故障诊断工具.md](./03-故障诊断工具.md)

本目录优先收录：

- 常见故障现象的排查路径
- 容器与镜像的调试操作
- 诊断命令和观测工具的使用方法

本目录不优先收录：

- 正常流程下的基础命令说明
- 生产部署策略
- 与专题无关的临时排障记录
