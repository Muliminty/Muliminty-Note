# Docker 基础入门目录说明

本目录用于建立 Docker 的最小可用认知与初始操作能力。

推荐阅读顺序：

1. [01-安装与环境配置.md](./01-安装与环境配置.md)
2. [02-核心概念.md](./02-核心概念.md)
3. [03-基本命令.md](./03-基本命令.md)

本目录优先收录：

- 本地安装与环境验证
- 镜像、容器、仓库等基础概念
- 常用 CLI 命令与最基本操作

本目录不优先收录：

- 镜像构建优化
- 多服务编排
- 生产环境部署与故障排查
