# Docker Compose 目录说明

本目录用于收纳 Docker Compose 相关的正式知识笔记，重点覆盖多服务项目的配置、编排与落地实践。

推荐阅读顺序：

1. [01-Compose基础.md](./01-Compose基础.md)
2. [04-Compose文件编写指南.md](./04-Compose文件编写指南.md)
3. [02-Compose高级特性.md](./02-Compose高级特性.md)
4. [03-多服务编排实践.md](./03-多服务编排实践.md)

本目录优先收录：

- Compose 安装与基本命令
- `docker-compose.yml` 文件结构与配置约束
- 环境变量、覆盖文件、健康检查等高级能力
- 多服务项目的编排案例

本目录不优先收录：

- 单容器基础命令
- 纯 Dockerfile 语法
- 与具体线上故障强绑定的排错记录
