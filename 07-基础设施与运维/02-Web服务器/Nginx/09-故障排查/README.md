# Nginx 故障排查目录说明

本目录用于收纳 Nginx 在配置、转发、部署、访问控制和运行状态中的常见问题排查主题。

优先收录：

- 配置语法错误与热更新失败
- 502、504、404、证书错误等常见问题
- 反向代理、静态资源、子路径部署相关排障
- 结合日志与监控的故障定位方法

不优先收录：

- 基础配置教程
- 仅具个人临时性的调试备忘
- 已经沉淀为稳定最佳实践的内容
