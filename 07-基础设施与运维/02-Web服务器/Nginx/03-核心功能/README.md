# Nginx 核心功能目录说明

本目录用于收纳 Nginx 在生产使用中最核心的能力型主题，包括虚拟主机、静态文件服务、反向代理与负载均衡。

推荐阅读顺序：

1. [虚拟主机配置.md](./虚拟主机配置.md)
2. [静态文件服务.md](./静态文件服务.md)
3. [反向代理.md](./反向代理.md)
4. [负载均衡.md](./负载均衡.md)

本目录优先收录：

- 站点接入与域名路由能力
- 静态文件承载与缓存控制
- 上游代理与请求转发能力
- 流量分发与高可用能力

本目录不优先收录：

- 安装与基础命令
- 全局语法说明
- 具体项目部署案例
