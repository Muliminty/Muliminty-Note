# Nginx 高级功能目录说明

本目录用于收纳超出基础接入能力之外的 Nginx 高阶能力主题。

优先收录：

- 高级缓存策略
- 灰度发布与流量控制
- 动态路由与复杂转发
- SSL/TLS、HTTP/2、HTTP/3 等进阶能力

不优先收录：

- 基础虚拟主机配置
- 基础反向代理配置
- 单项目部署案例
