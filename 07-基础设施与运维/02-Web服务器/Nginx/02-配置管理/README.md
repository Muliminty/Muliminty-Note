# Nginx 配置管理目录说明

本目录用于收纳 Nginx 配置文件组织、变量机制与配置发布相关笔记。

当前阅读顺序：

1. [01-配置文件结构.md](./01-配置文件结构.md)
2. [02-变量使用.md](./02-变量使用.md)
3. [03-配置检查与热更新.md](./03-配置检查与热更新.md)

本目录优先承载：

- 配置上下文与 include 组织方式
- 内置变量、自定义变量与 map 规则
- 配置验证、热更新、回滚与自动化发布

不优先承载：

- 反向代理具体业务配置
- 单一部署案例
- 与安全、性能直接相关的专题正文
