# Nginx 安全目录说明

本目录用于收纳 Nginx 与访问控制、TLS、安全头、限流和防护策略相关的主题。

推荐阅读顺序：

1. [01-HTTPS与证书配置.md](./01-HTTPS与证书配置.md)
2. [02-CORS与安全头配置.md](./02-CORS与安全头配置.md)
3. [03-限流与恶意请求防护.md](./03-限流与恶意请求防护.md)

优先收录：

- HTTPS 与证书配置
- 安全响应头与 CORS 控制
- 限流、防刷、防恶意请求
- 访问控制与敏感路径保护

不优先收录：

- 基础站点接入
- 业务级鉴权逻辑
- 纯性能调优内容
