# 客户端缓存策略

## 1. 关注点

- 拉取时机
- 失效策略
- 重新验证
- 乐观更新

## 2. 实践建议

- 把服务端状态与本地 UI 状态分开治理
- 缓存策略要服务于交互体验，而不是只追求命中率
