# GraphQL进阶

## 1. 进阶关注点

- 查询粒度设计
- 缓存与失效策略
- 订阅与实时更新
- 查询复杂度控制

## 2. 实践建议

- schema 设计优先服务真实业务边界
- 不要把 GraphQL 当成自动免设计方案
