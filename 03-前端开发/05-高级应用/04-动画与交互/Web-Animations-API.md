# Web-Animations-API

## 1. 这是什么

`Web Animations API` 是浏览器提供的原生动画接口，可用来以脚本方式定义和控制动画。

## 2. 适合场景

- 精细控制动画时间轴
- 动态创建和销毁动画
- 与浏览器原生能力更紧密协同
