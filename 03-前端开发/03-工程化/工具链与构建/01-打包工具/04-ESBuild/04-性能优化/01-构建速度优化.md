## 性能优势

### 为什么快？

1. **Go 语言实现**：编译型语言，执行速度快
2. **并行处理**：充分利用多核 CPU
3. **单一可执行文件**：启动速度快
4. **优化的算法**：高效的解析和转换

## 并行处理

### 自动并行

ESBuild 自动利用多核 CPU 进行并行处理，无需额外配置。

---

## 增量构建

### Watch 模式

```javascript
require('esbuild').context({
  entryPoints: ['src/index.js'],
  bundle: true,
  outfile: 'dist/bundle.js'
}).then(ctx => {
  ctx.watch()  // 增量构建
})
```
