# 核心概念

> Entry、Output、Format、Platform 等核心概念

---

## 📋 学习目标

- ✅ 理解 Entry（入口）概念
- ✅ 理解 Output（输出）配置
- ✅ 掌握 Format（输出格式）
- ✅ 理解 Platform（平台）选择
- ✅ 掌握 Target（目标环境）配置

---

## Entry（入口）

### 单入口

```javascript
{
  entryPoints: ['src/index.js']
}
```

### 多入口

```javascript
{
  entryPoints: {
    'main': 'src/index.js',
    'admin': 'src/admin.js'
  }
}
```

---

## Output（输出）

### 单文件输出

```javascript
{
  outfile: 'dist/bundle.js'
}
```

### 多文件输出

```javascript
{
  outdir: 'dist',
  entryNames: '[name]-[hash]'
}
```

---

## Format（输出格式）

### ESM（ES 模块）

```javascript
{
  format: 'esm'
}
```

### CommonJS

```javascript
{
  format: 'cjs'
}
```

### IIFE

```javascript
{
  format: 'iife',
  globalName: 'MyLibrary'
}
```

---

## Platform（平台）

### Browser（浏览器）

```javascript
{
  platform: 'browser'
}
```

### Node（Node.js）

```javascript
{
  platform: 'node'
}
```

---

## Target（目标环境）

```javascript
{
  target: 'es2020'  // 或 ['es2020', 'chrome80', 'firefox78']
}
```
