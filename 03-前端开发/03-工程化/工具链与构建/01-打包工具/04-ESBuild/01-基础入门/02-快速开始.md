# 快速开始

> 安装、第一个项目、基础配置

---

## 📋 学习目标

- ✅ 能够安装 ESBuild
- ✅ 创建第一个 ESBuild 项目
- ✅ 掌握基础配置
- ✅ 理解命令行和 API 使用

---

## 安装

### npm 安装

```bash
npm install --save-dev esbuild
```

### 全局安装

```bash
npm install -g esbuild
```

### 验证安装

```bash
npx esbuild --version
```

---

## 第一个项目

### 项目结构

```
my-project/
├── src/
│   └── index.js
├── package.json
└── esbuild.config.js
```

### 基础代码

**src/index.js**：
```javascript
console.log('Hello ESBuild!')
```

### 命令行构建

```bash
npx esbuild src/index.js --bundle --outfile=dist/bundle.js
```

### 使用配置文件

**esbuild.config.js**：
```javascript
require('esbuild').build({
  entryPoints: ['src/index.js'],
  bundle: true,
  outfile: 'dist/bundle.js',
}).catch(() => process.exit(1))
```

**运行**：
```bash
node esbuild.config.js
```

---

## 基础配置

### 常用选项

```javascript
require('esbuild').build({
  entryPoints: ['src/index.js'],
  bundle: true,
  outfile: 'dist/bundle.js',
  format: 'esm',        // 'esm' | 'cjs' | 'iife'
  platform: 'browser',  // 'browser' | 'node'
  target: 'es2020',     // 目标环境
  minify: true,         // 代码压缩
  sourcemap: true,      // Source Map
})
```
