## 完整配置选项

### 基础配置

```javascript
require('esbuild').build({
  // 入口
  entryPoints: ['src/index.js'],
  
  // 输出
  bundle: true,
  outfile: 'dist/bundle.js',
  // 或
  outdir: 'dist',
  
  // 格式和平台
  format: 'esm',
  platform: 'browser',
  target: 'es2020',
  
  // 压缩
  minify: true,
  
  // Source Map
  sourcemap: true,
  
  // 代码分割
  splitting: true,
  
  // 插件
  plugins: []
})
```

## 环境变量

### 使用环境变量

```javascript
const isProduction = process.env.NODE_ENV === 'production'

require('esbuild').build({
  minify: isProduction,
  sourcemap: !isProduction
})
```

### 注入环境变量

```javascript
{
  define: {
    'process.env.NODE_ENV': '"production"',
    'process.env.API_URL': '"https://api.example.com"'
  }
}
```

## 配置文件

### JavaScript 配置

**esbuild.config.js**：
```javascript
module.exports = {
  entryPoints: ['src/index.js'],
  bundle: true,
  outfile: 'dist/bundle.js'
}
```

### TypeScript 配置

**esbuild.config.ts**：
```typescript
import { BuildOptions } from 'esbuild'

const config: BuildOptions = {
  entryPoints: ['src/index.ts'],
  bundle: true,
  outfile: 'dist/bundle.js'
}

export default config
```
