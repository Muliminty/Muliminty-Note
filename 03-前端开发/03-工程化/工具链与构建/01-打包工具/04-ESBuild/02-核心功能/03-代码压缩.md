## 内置压缩

### 启用压缩

```javascript
{
  minify: true  // 启用所有压缩
}
```

### 分别控制

```javascript
{
  minifyWhitespace: true,   // 压缩空白
  minifyIdentifiers: true,  // 压缩标识符
  minifySyntax: true        // 压缩语法
}
```

---

## 压缩选项

### 压缩空白

```javascript
{
  minifyWhitespace: true
}
```

### 压缩标识符

```javascript
{
  minifyIdentifiers: true
}
```

### 压缩语法

```javascript
{
  minifySyntax: true
}
```
