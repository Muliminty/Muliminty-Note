## 插件基础

### 插件结构

```javascript
const myPlugin = {
  name: 'my-plugin',
  setup(build) {
    // 插件逻辑
  }
}
```

### 使用插件

```javascript
{
  plugins: [myPlugin]
}
```

---

## 插件钩子

### onResolve

解析模块路径：

```javascript
build.onResolve({ filter: /^env$/ }, args => ({
  path: args.path,
  namespace: 'env-ns'
}))
```

### onLoad

加载模块内容：

```javascript
build.onLoad({ filter: /.*/, namespace: 'env-ns' }, () => ({
  contents: JSON.stringify(process.env),
  loader: 'json'
}))
```

## 常用插件

### esbuild-plugin-alias

路径别名：

```javascript
const alias = require('esbuild-plugin-alias')

{
  plugins: [
    alias({
      'react': require.resolve('react')
    })
  ]
}
```

### esbuild-plugin-copy

复制文件：

```javascript
const copy = require('esbuild-plugin-copy')

{
  plugins: [
    copy({
      assets: {
        from: ['public/**/*'],
        to: ['dist']
      }
    })
  ]
}
```
