# 列表渲染

## 1. 一句话概括主题

列表渲染就是把一组数据映射成一组 React 元素，最常见的写法是 `array.map(...)`。

## 2. 核心原则

- 数据驱动 UI，不手写重复 JSX
- 每一项都要提供稳定的 `key`
- `key` 应优先使用业务主键，而不是数组索引

## 3. 常见风险

- 用索引做 `key`，导致重排时状态错位
- 在 `map` 中写过重逻辑，拉低可读性
- 渲染前不做空态处理，导致边界条件混乱
