# 组件组合

## 1. 一句话概括主题

组件组合就是把小组件按职责拼起来，而不是在一个大组件里堆所有逻辑。

## 2. 常见方式

- 父子嵌套
- `children` 透传
- 通过 Props 注入局部内容
- 容器组件与展示组件分层

## 3. 设计建议

- 让每个组件只负责一个稳定职责
- 把可变部分放进 Props，而不是复制组件
- 优先组合，少做过早抽象
