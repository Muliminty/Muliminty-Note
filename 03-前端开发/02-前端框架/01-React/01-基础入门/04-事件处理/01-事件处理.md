# 事件处理

## 1. 一句话概括主题

事件处理就是在 React 组件中响应点击、输入、提交等用户交互。

## 2. 基本原则

- 用函数作为事件处理器
- 需要参数时用闭包或包装函数传递
- 区分“渲染时执行”与“事件触发时执行”

## 3. 常见主题

- 事件绑定
- 事件对象
- 合成事件
- 阻止默认行为与阻止冒泡
