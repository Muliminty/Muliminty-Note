# 样式系统

## 1. 基本特点

- 样式使用对象表达
- 不支持传统浏览器 CSS 全量能力
- 布局核心仍是 `Flexbox`

## 2. 关注点

- 平台差异
- 阴影与字体支持差异
- 可复用样式抽象
