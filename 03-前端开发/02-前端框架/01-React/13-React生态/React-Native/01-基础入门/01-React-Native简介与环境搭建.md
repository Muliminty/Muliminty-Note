# React Native 简介与环境搭建

## 1. 这是什么

`React Native` 允许使用 `JavaScript / TypeScript + React` 语法开发 `iOS` 与 `Android` 应用。

## 2. 环境组成

- `Node.js`
- `JDK`
- `Android Studio`
- `Xcode`
- 真机或模拟器

## 3. 搭建建议

- 先确定使用 `Expo` 还是原生工作流
- 环境搭建完成后先跑官方模板项目
- 真机调试链路应尽早打通
