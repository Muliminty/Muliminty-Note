# 项目结构与配置

## 1. 常见目录

- `src`：业务代码
- `android / ios`：平台原生工程
- `assets`：图片、字体等资源

## 2. 关键配置

- Babel 配置
- TypeScript 配置
- Metro 配置
- 环境变量管理

## 3. 建议

- 先划清页面、组件、服务与状态目录边界
- 原生平台配置变更要同步记录
