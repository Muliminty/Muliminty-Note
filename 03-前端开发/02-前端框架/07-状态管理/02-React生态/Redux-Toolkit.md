# Redux Toolkit

`Redux Toolkit` 是官方推荐的 `Redux` 现代写法，重点解决样板代码过多的问题。

## 核心能力

- `slice`
- `configureStore`
- 内置不可变更新支持
- 与异步流程工具更易配合
