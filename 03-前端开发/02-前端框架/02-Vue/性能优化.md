# 性能优化

## 1. 优化关注点

- 组件是否被不必要地重新渲染
- 大列表、频繁更新和异步加载是否拆分得当
- 状态粒度是否过粗

## 2. 常见手段

- 使用异步组件与懒加载降低首屏压力
- 避免在模板中堆叠高成本表达式
- 为长列表引入虚拟列表思路
- 区分一次性数据与持续响应式数据

## 3. 工程化配合

- 代码分割与按需加载
- 静态资源压缩与缓存
- 构建结果分析
