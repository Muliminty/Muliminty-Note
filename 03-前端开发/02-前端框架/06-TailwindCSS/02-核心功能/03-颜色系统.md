# 颜色系统

## 1. 核心内容

- `bg-*`：背景色
- `text-*`：文字色
- `border-*`：边框色
- `opacity-*`：透明度控制

## 2. 实践建议

- 颜色命名尽量贴近设计 token
- 主色、辅助色、状态色要有明确边界
- 深浅层级要服务于信息层次，而不是随意堆砌
