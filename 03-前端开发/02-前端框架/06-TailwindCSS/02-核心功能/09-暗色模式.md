# 暗色模式

## 1. 核心点

- 通过 `dark:` 变体定义暗色样式
- 可基于系统偏好或业务开关切换主题
- 颜色对比度要重新校验

## 2. 实践建议

- 优先用设计 token 管理明暗主题
- 避免只反转颜色，不处理层级和可读性
