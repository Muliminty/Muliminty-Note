## 🎯 学习路径

### 阶段一：基础入门

**目标**：掌握 Java 基础语法和面向对象编程

**学习顺序**：
1. [Java 简介与特点](./01-基础入门/Java-简介与特点.md) — 了解 Java 语言
2. [安装与环境配置](./01-基础入门/安装与环境配置.md) — 搭建开发环境
3. [基础语法](./01-基础入门/基础语法.md) — 学习基础语法
4. [数据类型与变量](./01-基础入门/数据类型与变量.md) — 掌握数据类型
5. [控制结构](./01-基础入门/控制结构.md) — 学习条件语句和循环
6. [函数](./01-基础入门/函数.md) — 学习方法的定义和使用
7. [类与对象](./01-基础入门/类与对象.md) — 理解面向对象基础
8. [继承与多态](./01-基础入门/继承与多态.md) — 掌握面向对象核心特性
9. [接口与抽象类](./01-基础入门/接口与抽象类.md) — 学习接口和抽象类
10. [集合框架](./02-核心特性/集合框架.md) — 掌握集合的使用

**学习检查点**：
- ✅ 能够编写简单的 Java 程序
- ✅ 理解面向对象的基本概念
- ✅ 能够使用集合框架处理数据
