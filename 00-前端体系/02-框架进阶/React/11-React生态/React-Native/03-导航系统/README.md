# 导航系统

> 掌握 React Native 应用的路由和导航

## 学习目标

- 掌握 React Navigation 的安装和配置
- 理解不同类型的导航器及其使用场景
- 能够实现复杂的导航流程和参数传递

## 目录结构

- [01-React-Navigation基础](./01-React-Navigation基础.md) — 安装配置、基本使用
- [02-导航器类型](./02-导航器类型.md) — Stack、Tab、Drawer 导航器
- [03-导航参数传递](./03-导航参数传递.md) — 路由参数、导航选项、深度链接

## 学习建议

1. 从 Stack Navigator 开始，这是最常用的导航器
2. 根据应用需求选择合适的导航器组合
3. 掌握导航参数传递，实现页面间数据共享
4. 完成实践项目：实现一个多页面的应用导航
