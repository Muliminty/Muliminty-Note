# 工程化实践

> 掌握 React Native 开发工具和最佳实践

## 学习目标

- 能够配置完整的开发工具链
- 掌握调试技巧和工具使用
- 建立测试策略和代码质量保障

## 目录结构

- [01-开发工具配置](./01-开发工具配置.md) — ESLint、Prettier、TypeScript 配置
- [02-调试技巧](./02-调试技巧.md) — React Native Debugger、Flipper、日志调试
- [03-测试策略](./03-测试策略.md) — Jest、React Native Testing Library

## 学习建议

1. 从一开始就配置好开发工具，提高开发效率
2. 掌握多种调试方法，快速定位问题
3. 建立测试习惯，保证代码质量
4. 完成实践项目：为一个现有项目配置完整的工具链
