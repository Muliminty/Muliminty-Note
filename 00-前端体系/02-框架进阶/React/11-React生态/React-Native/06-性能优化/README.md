# 性能优化

> 掌握 React Native 应用的性能优化技巧

## 学习目标

- 能够识别性能瓶颈
- 掌握渲染性能优化技巧
- 理解内存管理和图片优化策略

## 目录结构

- [01-渲染性能优化](./01-渲染性能优化.md) — FlatList、优化重渲染、Memo 使用
- [02-内存管理](./02-内存管理.md) — 内存泄漏排查、图片缓存
- [03-图片优化](./03-图片优化.md) — 图片压缩、懒加载、缓存策略

## 学习建议

1. 使用性能分析工具（如 React DevTools）识别问题
2. 优先优化列表渲染性能（使用 FlatList）
3. 注意内存泄漏，及时清理监听器和定时器
4. 完成实践项目：优化一个性能较差的现有应用
