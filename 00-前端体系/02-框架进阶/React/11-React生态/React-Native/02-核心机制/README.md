# 核心机制

> 理解 React Native 的工作原理和与 React 的差异

## 学习目标

- 理解 React Native 的渲染机制
- 掌握 React Native 与 React 的差异
- 了解 JavaScript 与原生代码的通信机制

## 目录结构

- [01-渲染机制](./01-渲染机制.md) — 原生组件渲染、Bridge 通信
- [02-与React的差异](./02-与React的差异.md) — 组件差异、API 差异、样式差异
- [03-原生模块通信](./03-原生模块通信.md) — JavaScript 与原生代码通信机制

## 学习建议

1. 深入理解 React Native 的架构设计
2. 对比 React 和 React Native 的差异，避免混淆
3. 理解 Bridge 通信机制，为后续原生模块开发打基础
