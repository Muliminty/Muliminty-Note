# 状态管理

> 掌握 React Native 应用的状态管理方案

## 学习目标

- 了解不同状态管理方案的适用场景
- 掌握 Context API 在 React Native 中的应用
- 能够集成 Redux 或 Zustand 进行状态管理

## 目录结构

- [01-Context-API在RN中的应用](./01-Context-API在RN中的应用.md) — 使用 Context 管理全局状态
- [02-Redux在RN中的应用](./02-Redux在RN中的应用.md) — Redux Toolkit 集成
- [03-Zustand在RN中的应用](./03-Zustand在RN中的应用.md) — 轻量级状态管理

## 学习建议

1. 对于简单应用，优先使用 Context API
2. 对于复杂应用，考虑使用 Redux 或 Zustand
3. 根据项目规模和团队经验选择合适的状态管理方案
4. 完成实践项目：使用状态管理实现用户认证和数据持久化
