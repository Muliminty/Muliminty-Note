# 基础入门

> 掌握 React Native 核心概念，能够搭建开发环境并创建第一个应用

## 学习目标

- 了解 React Native 的基本概念和特点
- 能够搭建 React Native 开发环境
- 掌握核心组件和 API 的使用
- 理解 React Native 的样式系统和布局

## 目录结构

- [01-React-Native简介与环境搭建](./01-React-Native简介与环境搭建.md) — 环境搭建、开发工具配置
- [02-项目结构与配置](./02-项目结构与配置.md) — 项目结构、配置文件详解
- [03-核心组件与API](./03-核心组件与API.md) — View、Text、Image、ScrollView 等核心组件
- [04-样式系统](./04-样式系统.md) — StyleSheet、Flexbox 布局
- [05-布局系统](./05-布局系统.md) — Flexbox 在 React Native 中的应用

## 学习建议

1. 先完成环境搭建，确保能够运行示例应用
2. 逐个学习核心组件，理解每个组件的用途和属性
3. 重点掌握 Flexbox 布局，这是 React Native 布局的基础
4. 完成实践项目：创建一个简单的待办事项应用
