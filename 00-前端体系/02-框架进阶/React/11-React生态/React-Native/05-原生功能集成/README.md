# 原生功能集成

> 掌握原生模块开发和平台特定功能

## 学习目标

- 能够创建和使用原生模块
- 掌握平台特定代码的编写
- 能够集成设备 API（相机、定位、通知等）

## 目录结构

- [01-原生模块开发](./01-原生模块开发.md) — 创建原生模块、桥接原生代码
- [02-平台特定代码](./02-平台特定代码.md) — Platform API、平台特定文件
- [03-设备API使用](./03-设备API使用.md) — 相机、定位、通知等设备功能

## 学习建议

1. 优先使用社区已有的原生模块库
2. 只有在必要时才自己开发原生模块
3. 理解平台差异，编写平台特定代码
4. 完成实践项目：集成相机和定位功能
