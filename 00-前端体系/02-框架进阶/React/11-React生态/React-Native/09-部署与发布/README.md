# 部署与发布

> 掌握应用的打包和发布流程

## 学习目标

- 能够配置 Android 和 iOS 的打包环境
- 掌握应用签名和证书配置
- 了解应用商店发布流程

## 目录结构

- [01-Android打包发布](./01-Android打包发布.md) — 签名配置、APK/AAB 打包
- [02-iOS打包发布](./02-iOS打包发布.md) — 证书配置、Archive 打包
- [03-应用商店发布](./03-应用商店发布.md) — Google Play、App Store 发布流程

## 学习建议

1. 提前了解应用商店的审核规则
2. 配置好签名和证书，避免发布时的问题
3. 准备应用截图、描述等素材
4. 完成实践项目：将一个应用打包并发布到测试环境
