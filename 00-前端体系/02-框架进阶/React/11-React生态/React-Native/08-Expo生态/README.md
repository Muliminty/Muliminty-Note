# Expo 生态

> 掌握 Expo 工具链的使用

## 学习目标

- 了解 Expo 的优势和适用场景
- 掌握 Expo CLI 和 Expo Go 的使用
- 能够使用 Expo SDK 开发应用

## 目录结构

- [01-Expo简介与使用](./01-Expo简介与使用.md) — Expo CLI、Expo Go、开发流程
- [02-Expo-SDK使用](./02-Expo-SDK使用.md) — 常用 SDK API、相机、定位等
- [03-Expo-开发流程](./03-Expo-开发流程.md) — 开发、构建、发布流程

## 学习建议

1. 对于新项目，优先考虑使用 Expo
2. 理解 Expo 的限制，必要时使用 bare workflow
3. 充分利用 Expo SDK 提供的功能
4. 完成实践项目：使用 Expo 创建一个完整的应用
