# Tailwind CSS 目录说明

本目录收纳 `Tailwind CSS` 相关的正式专题内容。

## 当前角色

- `!MOC-TailwindCSS.md`：主题导航入口，负责阅读路径和专题索引。
- `README.md`：目录说明文件，负责说明目录结构与维护边界。
- 子目录正文：承载基础入门、核心功能与实战应用等具体内容。


## 维护约束

- 导航职责统一由 [!MOC-TailwindCSS.md](./!MOC-TailwindCSS.md) 承担。
