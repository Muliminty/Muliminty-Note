## 🎯 快速参考

### 常用工具类速查

#### 布局
- **Flexbox**: `flex`, `flex-col`, `items-center`, `justify-between`
- **Grid**: `grid`, `grid-cols-3`, `gap-4`
- **定位**: `relative`, `absolute`, `fixed`, `sticky`

#### 间距
- **内边距**: `p-4`, `px-6`, `py-4`
- **外边距**: `m-2`, `mx-auto`, `my-8`

#### 颜色
- **背景**: `bg-blue-500`, `bg-gray-100`
- **文字**: `text-white`, `text-gray-600`
- **边框**: `border`, `border-gray-300`

#### 响应式
- **断点**: `sm:`, `md:`, `lg:`, `xl:`, `2xl:`
- **示例**: `text-sm md:text-base lg:text-lg`

#### 状态变体
- **交互**: `hover:bg-blue-600`, `focus:ring-2`, `active:scale-95`
- **状态**: `disabled:opacity-50`, `group-hover:`

## 🚀 快速开始

### 5 分钟快速上手

1. **安装 Tailwind CSS**
   ```bash
   npm install -D tailwindcss
   npx tailwindcss init
   ```

2. **配置 tailwind.config.js**
   ```js
   module.exports = {
     content: ['./src/**/*.{js,jsx,ts,tsx}'],
     theme: {
       extend: {},
     },
     plugins: [],
   }
   ```

3. **添加 Tailwind 指令**
   ```css
   @tailwind base;
   @tailwind components;
   @tailwind utilities;
   ```

4. **开始使用**
   ```html
   <div class="bg-blue-500 text-white p-4 rounded-lg">
     你好，Tailwind CSS！
   </div>
   ```

详细步骤请参考：[快速开始](./01-基础入门/02-快速开始.md)
