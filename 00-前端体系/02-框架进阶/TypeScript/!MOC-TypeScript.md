# TypeScript（TypeScript）MOC

> TypeScript 是 JavaScript 的超集，添加了静态类型系统。
> 
> **学习路径**：学习 TypeScript 前需要掌握 [JavaScript 基础](../../01-基础入门/JavaScript/!MOC-javascript.md)。TypeScript 常与前端框架（[React](../React/!MOC-React.md)、[Vue](../Vue/!MOC-Vue.md)、[Angular](../Angular/!MOC-Angular.md)）配合使用。

## 📚 核心主题

- [编译时类型设计](./编译时类型设计.md) — 类型系统设计（编译时检查，详见 [Babel 转换管线](../../03-工程化实践/工具链与构建/02-编译工具/Babel转换管线.md)）
- [运行时类型保护与模式](./运行时类型保护与模式.md) — 类型安全实践（配合 [测试](../../04-质量保障/测试/!MOC-测试.md) 使用）

## 🎯 核心概念

### 基础类型
- 原始类型
- 数组和元组
- 枚举
- 任意类型

### 类型系统
- 接口（Interface）
- 类型别名（Type Alias）
- 联合类型和交叉类型
- 泛型

### 高级类型
- 映射类型
- 条件类型
- 工具类型
- 模板字面量类型

### 类型推断
- 类型推断规则
- 类型断言（详见 [静态分析与类型安全](../../04-质量保障/测试/静态分析与类型安全.md)）
- 类型守卫
- 类型缩小

## 📝 学习建议

1. **前置知识**：必须掌握 [JavaScript 基础](../../01-基础入门/JavaScript/!MOC-javascript.md)
2. **学习顺序**：JavaScript → TypeScript → 前端框架（[React](../React/!MOC-React.md)、[Vue](../Vue/!MOC-Vue.md)、[Angular](../Angular/!MOC-Angular.md)）
3. **实践应用**：
   - 与前端框架结合使用，提升代码质量和开发体验
   - 配合 [测试](../../04-质量保障/测试/!MOC-测试.md) 工具使用
   - 使用 [Babel](../../03-工程化实践/工具链与构建/02-编译工具/Babel转换管线.md) 或 TypeScript Compiler 编译
4. **框架选择**：
   - [React](../React/!MOC-React.md) + TypeScript：主流选择
   - [Vue](../Vue/!MOC-Vue.md) + TypeScript：Vue 3 对 TypeScript 支持良好
   - [Angular](../Angular/!MOC-Angular.md)：基于 TypeScript，必须学习
