# HTML MOC

> HTML（HyperText Markup Language）是网页的骨架，定义网页的结构和内容。
> 
> **学习路径**：HTML 是前端三大基础之一，通常与 [CSS](../CSS/!MOC-CSS.md) 和 [JavaScript](../JavaScript/!MOC-javascript.md) 配合使用，共同构建完整的网页应用。

---

## 📚 基础概念

- [语义化HTML](./语义化HTML.md) — 语义化标签与最佳实践（与 [CSS 布局](../CSS/!MOC-CSS.md) 配合使用）
- [表单与验证](./表单与验证.md) — 表单设计与验证规则（通常需要 [JavaScript](../JavaScript/!MOC-javascript.md) 进行交互处理）
- [无障碍基础-ARIA入门](./无障碍基础-ARIA入门.md) — 可访问性基础（详见 [无障碍 MOC](../../04-质量保障/无障碍/!MOC-无障碍.md)）

---

## 🎯 核心主题

### HTML5 新特性
- 语义化标签
- 表单增强
- 媒体元素（audio、video）
- Canvas 和 SVG

### 表单处理
- 表单元素
- 输入验证
- 表单提交
- 文件上传

### 可访问性
- ARIA 属性（深入学习：[无障碍 MOC](../../04-质量保障/无障碍/!MOC-无障碍.md)）
- 语义化标签
- 键盘导航
- 屏幕阅读器支持

---

## 📝 学习建议

1. **学习顺序**：先掌握 HTML 基础，然后学习 [CSS](../CSS/!MOC-CSS.md) 样式化，最后用 [JavaScript](../JavaScript/!MOC-javascript.md) 添加交互
2. **实践应用**：HTML 与 CSS 配合实现布局，与 JavaScript 配合实现动态交互
3. **进阶学习**：掌握 HTML 后，可以学习前端框架（[React](../../02-框架进阶/React/!MOC-React.md)、[Vue](../../02-框架进阶/Vue/!MOC-Vue.md) 等）
4. **可访问性**：从 HTML 设计阶段就要考虑可访问性（详见 [无障碍 MOC](../../04-质量保障/无障碍/!MOC-无障碍.md)）
