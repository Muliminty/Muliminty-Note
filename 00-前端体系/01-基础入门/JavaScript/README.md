## 🎯 学习路径

1. **基础入门**：语言核心 → 内置对象与 API → 对象与类
2. **进阶学习**：模块化 → 异步编程 → 函数式编程
3. **实践应用**：浏览器环境与 DOM → 性能优化
4. **工程化**：工具与生态 → 测试与规范
