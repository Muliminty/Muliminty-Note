# CSS（CSS）MOC

> CSS（Cascading Style Sheets）是网页的样式表语言，控制网页的外观和布局。
> 
> **学习路径**：CSS 与 [HTML](../HTML/!MOC-HTML.md) 配合使用，样式化网页元素。掌握 CSS 后，可以学习 [JavaScript](../JavaScript/!MOC-javascript.md) 实现动态样式和交互效果。

- [响应式设计/移动优先](./响应式设计-移动优先.md) — 响应式设计原理与实践（详见 [移动与跨平台 MOC](../../05-高级应用/移动与跨平台/!MOC-移动与跨平台.md)）

## 📝 学习建议

1. **前置知识**：需要先掌握 [HTML 基础](../HTML/!MOC-HTML.md)，了解网页结构
2. **学习顺序**：HTML → CSS → JavaScript，循序渐进
3. **实践应用**：CSS 与 [JavaScript](../JavaScript/!MOC-javascript.md) 配合实现动态样式和交互
4. **进阶学习**：掌握 CSS 后，可以学习前端框架，或深入学习 [设计系统](../../05-高级应用/架构/设计系统/!MOC-设计系统.md)
