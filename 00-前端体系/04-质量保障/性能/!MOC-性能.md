# 性能优化（Performance）MOC

> 前端性能优化策略和工具，包括加载性能、运行时性能、资源优化。
> 
> **学习路径**：性能优化贯穿整个开发流程，与 [构建工具](../../03-工程化实践/工具链与构建/!MOC-工具链与构建.md) 配合实现代码优化，与 [测试](../测试/!MOC-测试.md) 配合验证优化效果。

---

## 📚 核心主题

- [加载性能（LCP/FCP/TTI）](./加载性能.md) — 页面加载性能优化（配合 [CSS 优化](../../01-基础入门/CSS/性能与关键渲染路径.md)）
- [运行时性能（FPS/长任务）](./运行时性能.md) — 运行时性能优化（针对 [React](../../02-框架进阶/React/性能优化.md) 和 [Vue](../../02-框架进阶/Vue/性能优化.md) 框架）
- [关键资源优化（图片/字体/预加载）](./关键资源优化.md) — 资源优化策略
- [代码分割与按需加载](./代码分割与按需加载.md) — 代码分割方案（配合 [构建工具](../../03-工程化实践/工具链与构建/!MOC-工具链与构建.md) 和 [工程化](../../03-工程化实践/工程化/模块化与分包.md)）

## 🎯 性能指标

### 加载性能
- LCP（Largest Contentful Paint）
- FCP（First Contentful Paint）
- TTI（Time to Interactive）
- CLS（Cumulative Layout Shift）

### 运行时性能
- FPS（Frames Per Second）
- 长任务（Long Tasks）
- 内存泄漏
- 渲染性能

### 资源优化
- 图片优化
- 字体优化（配合 [CSS 优化](../../01-基础入门/CSS/性能与关键渲染路径.md)）
- 代码压缩（配合 [构建工具](../../03-工程化实践/工具链与构建/!MOC-工具链与构建.md)）
- 缓存策略（详见 [网络与传输](../../05-高级应用/网络与传输/缓存策略.md)）

## 📝 学习建议

1. **前置知识**：需要掌握前端基础（[HTML](../../01-基础入门/HTML/!MOC-HTML.md)、[CSS](../../01-基础入门/CSS/!MOC-CSS.md)、[JavaScript](../../01-基础入门/JavaScript/!MOC-javascript.md)）和前端框架
2. **学习顺序**：基础 → 框架 → 性能优化 → [测试](../测试/!MOC-测试.md) 验证
3. **优化维度**：
   - 加载性能：配合 [构建工具](../../03-工程化实践/工具链与构建/!MOC-工具链与构建.md) 优化
   - 运行时性能：配合框架优化（[React](../../02-框架进阶/React/性能优化.md)、[Vue](../../02-框架进阶/Vue/性能优化.md)）
   - 资源优化：配合 [网络与传输](../../05-高级应用/网络与传输/!MOC-网络与传输.md) 优化
4. **质量保障**：性能优化与 [测试](../测试/!MOC-测试.md) 和 [安全](../安全/!MOC-前端安全.md) 共同保障代码质量
