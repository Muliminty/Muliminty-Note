# 编程语言目录说明

本目录用于承载语言层知识，不再混放框架、部署、项目案例与排错内容。

优先收纳：

- 语言语法
- 类型系统
- 标准库与运行时
- 语言级工程实践
- 运行时与语言生态

当前收录：

- `JavaScript`
- `TypeScript`
- `Go`
- `Java`
- `Node.js`

边界说明：

- 语言基础、集合/JVM/并发基础、模块系统、事件循环、核心模块等内容放在本目录
- Web 框架使用、REST API、数据库访问模式、认证授权、微服务与服务治理放在 `04-服务端开发`

迁移来源：

- `00-前端体系/01-基础入门`
- `03-服务端/01-服务端语言`
