# 安装与版本管理（nvm）

Node.js 的安装和版本管理工具。

---

## 📚 安装 Node.js

### 方式一：官方安装包

1. 访问 [Node.js 官网](https://nodejs.org/)
2. 下载 LTS 版本（推荐）
3. 运行安装程序
4. 验证安装：

```bash
node --version
npm --version
```

### 方式二：使用 nvm（推荐）

nvm（Node Version Manager）可以管理多个 Node.js 版本。

#### macOS/Linux 安装 nvm

```bash
# 使用 curl
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash

# 或使用 wget
wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.0/install.sh | bash
```

#### Windows 安装 nvm

下载 [nvm-windows](https://github.com/coreybutler/nvm-windows/releases)

---

## 🎯 使用 nvm

### 安装 Node.js 版本

```bash
# 安装最新 LTS 版本
nvm install --lts

# 安装特定版本
nvm install 20.10.0
nvm install 18.19.0
```

### 切换版本

```bash
# 使用特定版本
nvm use 20.10.0

# 使用最新版本
nvm use node

# 使用 LTS 版本
nvm use --lts
```

### 查看已安装版本

```bash
# 列出已安装的版本
nvm list

# 列出所有可用版本
nvm list-remote
```

### 设置默认版本

```bash
# 设置默认版本
nvm alias default 20.10.0
```

## 💡 最佳实践

### 1. 使用 LTS 版本

```bash
# 生产环境使用 LTS
nvm install --lts
nvm use --lts
```

### 2. 项目指定版本

创建 `.nvmrc` 文件：

```bash
# .nvmrc
20.10.0
```

使用：

```bash
nvm use  # 自动使用 .nvmrc 中的版本
```

### 3. package.json engines

```json
{
  "engines": {
    "node": ">=18.0.0",
    "npm": ">=9.0.0"
  }
}
```

## 🔧 常见问题

### nvm 命令未找到

```bash
# 添加到 ~/.bashrc 或 ~/.zshrc
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"
```

### 权限问题

```bash
# 使用 nvm 安装的 Node.js 不需要 sudo
nvm install node
```
