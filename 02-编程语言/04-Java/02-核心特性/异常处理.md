# 异常处理

## 1. 异常体系

Java 中的异常体系主要分为：

- Checked Exception
- Unchecked Exception
- Error

## 2. 常见处理方式

- `try-catch`
- `throws`
- `finally`
- `try-with-resources`

## 3. 实践建议

- 在边界层统一转换异常语义
- 避免过度捕获顶层 `Exception`
- 资源关闭优先使用 `try-with-resources`
