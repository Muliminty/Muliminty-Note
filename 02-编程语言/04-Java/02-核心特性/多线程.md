# 多线程

## 1. 核心主题

- 线程创建与生命周期
- 共享变量与可见性
- `synchronized` 与锁机制
- `java.util.concurrent` 并发工具

## 2. 常见能力

- `Thread`
- `Runnable`
- `Callable`
- `ExecutorService`
- `Future` / `CompletableFuture`

## 3. 使用建议

- 优先使用线程池管理并发任务
- 明确线程安全边界
- 将并发模型与业务逻辑解耦
