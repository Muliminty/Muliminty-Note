# 系统规范导航

`99-系统` 目录用于维护整个知识库的治理规则、迁移方案与模板资产。

为减少规范文件横向铺开带来的理解成本，当前系统层统一收敛为以下主文件：

- [01-百科写作规范.md](./01-百科写作规范.md)
  统一约束 Markdown 语法、Frontmatter、命名规则与受控词表。
- [02-百科组织规范.md](./02-百科组织规范.md)
  统一约束一级目录、专题树、`!MOC`、`README.md`、`98-知识库` 与 `99-参考与归档` 的角色边界。
- [03-百科迁移方案.md](./03-百科迁移方案.md)
  说明个人百科目标结构、旧一级目录迁移方向与分阶段推进策略。
- [04-重构TODO.md](./04-重构TODO.md)
  作为长期迁移执行清单，记录阶段进度与待办事项。

辅助目录：

- [模板](./模板/00-模板使用指南.md)
  存放概念理解类、实战应用类、问题排错类模板。
- [99-参考与归档](./99-参考与归档/)
  存放旧版拆分规范、历史方案与已退役但仍需保留参考价值的系统文件。

使用建议：

1. 新增或重写知识文档前，优先阅读 [01-百科写作规范.md](./01-百科写作规范.md)。
2. 规划目录归属、判断是否拆专题时，优先阅读 [02-百科组织规范.md](./02-百科组织规范.md)。
3. 涉及全库迁移与一级目录调整时，优先阅读 [03-百科迁移方案.md](./03-百科迁移方案.md)。
4. 具体执行时，以 [04-重构TODO.md](./04-重构TODO.md) 为唯一进度清单。
