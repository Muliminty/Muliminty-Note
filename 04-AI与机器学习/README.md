## 📚 学习内容概览

### 01-基础理论

- AI 基础概念
- 数学基础（线性代数、概率论、微积分）

### 02-算法与模型

- 监督学习
- 无监督学习
- 深度学习
- 强化学习

### 03-框架与工具

- Python 机器学习框架
- 深度学习框架（TensorFlow、PyTorch）
- 模型部署与优化
- MLOps 工具

### 04-应用领域

- 自然语言处理（NLP）
- 计算机视觉（CV）
- 推荐系统
- 语音处理
- 时间序列分析

### 05-AI开发工具

- Prompt 工程
- AI 辅助开发（Cursor、Copilot）
- AI API 与 SDK
- **AI 应用开发**
  - **MCP 协议** - Model Context Protocol（AI 领域的 USB-C 接口）
  - **MCP 服务器开发** - 开发自定义 MCP Server
  - **MCP 配置与部署** - 在不同平台配置和部署 MCP
  - **MCP 最佳实践** - 安全性、性能优化与测试
