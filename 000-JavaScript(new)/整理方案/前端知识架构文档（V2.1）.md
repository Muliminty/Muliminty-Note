## 1 原始出处（溯源）
- TC39 Stage 4 提案
- 链接：https://tc39.es/ecma262/#sec-optional-chaining-operator

## 2 核心结论
> 可选链 `?.` 避免访问 null/undefined 时抛错。

## 3 最小可运行示例
```js
const obj = { a: { b: 1 } };
console.log(obj.a?.b); // 1
console.log(obj.x?.y); // undefined
````

## 4 应用场景

- API 返回数据不确定时
    
- 深层嵌套对象访问
