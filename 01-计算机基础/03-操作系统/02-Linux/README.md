# Linux 基础（Linux Fundamentals）

> Linux 系统入门与实践，掌握 Linux 系统的基本操作和管理

## 📚 核心内容

### 1. Linux 基础命令
- 文件操作命令（ls、cd、mkdir、rm 等）
- 文本处理命令（cat、grep、sed、awk）
- 系统信息命令（ps、top、df、du）
- 网络命令（ifconfig、netstat、ping）
- 压缩解压命令（tar、gzip、zip）

### 2. 文件系统
- Linux 文件系统结构
- 目录结构（/bin、/etc、/home、/var 等）
- 文件权限（rwx）
- 文件链接（硬链接、软链接）
- 文件系统类型（ext4、xfs、btrfs）

### 3. 进程管理
- 进程查看（ps、top、htop）
- 进程控制（kill、killall、pkill）
- 后台进程（&、nohup、screen、tmux）
- 进程监控与调试
- 系统资源监控

### 4. 权限管理
- 用户与用户组
- 文件权限（chmod、chown）
- 权限数字表示（755、644）
- sudo 与 su
- ACL（访问控制列表）

### 5. Shell 脚本
- Bash 基础语法
- 变量与数组
- 条件判断与循环
- 函数定义
- 脚本调试
- 正则表达式
- [Shell 配置文件](Shell%20配置文件.md)
- [zshrc配置详解](zshrc配置详解.md)
