# 现代网络技术（Modern Network Technologies）

> 理解当代网络架构和新兴技术，掌握软件定义网络、边缘计算、5G等先进网络技术

---

## 📚 核心内容

现代网络技术正在重塑网络架构和通信方式，从传统的硬件定义网络向软件定义、智能化、分布式网络发展。

### 1. 软件定义网络（SDN）

SDN是一种将网络控制平面与数据平面分离的网络架构，实现了网络的可编程化。

- [SDN概述](./01-软件定义网络.md) — SDN架构与基本概念
- [OpenFlow协议](./02-OpenFlow协议.md) — SDN南向接口标准
- [SDN控制器](./03-SDN控制器.md) — SDN网络大脑
- [SDN应用](./04-SDN应用.md) — SDN实际应用场景

### 2. 网络功能虚拟化（NFV）

NFV是将网络功能从专用硬件转移到虚拟化环境的技术。

- [NFV概述](./05-网络功能虚拟化.md) — NFV架构与组件
- [VNF与MANO](./06-VNF与MANO.md) — 虚拟网络功能与管理
- [NFV编排](./07-NFV编排.md) — 服务链与生命周期管理
- [NFV应用场景](./08-NFV应用场景.md) — NFV部署模式与案例

### 3. 5G网络技术

5G是第五代移动通信技术，不仅提供更快的速度，还支持大规模物联网应用。

- [5G网络概述](./09-5G网络概述.md) — 5G技术特点与应用场景
- [5G网络架构](./10-5G网络架构.md) — 5G网络架构与关键技术
- [边缘计算](./11-边缘计算.md) — MEC与边缘网络架构
- [网络切片](./12-网络切片.md) — 5G网络切片技术

### 4. 边缘计算网络

边缘计算将计算和存储资源部署在网络边缘，靠近终端用户。

- [边缘计算网络](./13-边缘计算网络.md) — 边缘网络架构与通信
- [雾计算](./14-雾计算.md) — 雾计算架构与应用

### 5. 物联网网络

物联网网络连接大量智能设备，提供无处不在的连接。
