# 网络基础

> 理解网络的基本概念和分层模型，掌握网络协议的基础知识

## 📚 核心内容

### 1. 网络概述
- 计算机网络的定义
- 网络的发展历史

- 网络拓扑结构

### 2. OSI 模型
- 七层网络模型

- 数据封装与解封装

- 网络层协议（IP、ICMP）
- 链路层协议

### 4. 网络设备
- 路由器（Router）
- 交换机（Switch）
- 集线器（Hub）
- 网桥（Bridge）
- 网关（Gateway）
