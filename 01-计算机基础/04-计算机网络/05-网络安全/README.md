# 网络安全（Network Security）

> 理解网络安全的基本原理和防护措施，掌握常见的网络攻击与防御技术

## 📚 核心内容

### 1. 网络安全基础

网络安全是保护计算机网络免受攻击、破坏或未经授权访问的技术和实践。了解网络安全的基础概念是构建安全网络环境的前提。

- [网络安全概述](./01-网络安全基础.md) — 网络安全的基本概念和重要性
- [威胁模型](./02-威胁模型.md) — 常见网络威胁与分类
- [安全架构](./03-安全架构.md) — 分层安全架构设计

### 2. 密码学基础

密码学是网络安全的基石，提供数据加密、身份验证和完整性保护。

- [对称加密](./04-对称加密.md) — AES、DES等对称加密算法
- [非对称加密](./05-非对称加密.md) — RSA、ECC等非对称加密算法
- [哈希函数](./06-哈希函数.md) — MD5、SHA系列哈希算法
- [数字签名](./07-数字签名.md) — 数字签名原理与应用

### 3. 网络防护技术

网络防护技术是保护网络资源免受攻击的具体措施和工具。

- [防火墙技术](./08-防火墙技术.md) — 包过滤、状态检测、应用层防火墙
- [入侵检测系统](./09-入侵检测系统.md) — IDS与IPS原理与部署
- [VPN技术](./10-VPN技术.md) — VPN协议与实现方式
- [网络访问控制](./11-网络访问控制.md) — NAC与身份验证

### 4. 安全协议

安全协议提供网络通信的安全保障，确保数据传输的机密性、完整性和可用性。

- [SSL/TLS协议](./12-SSL-TLS协议.md) — 安全传输层协议
- [IPSec协议](./13-IPSec协议.md) — 网络层安全协议
- [SSH协议](./14-SSH协议.md) — 安全远程访问协议
