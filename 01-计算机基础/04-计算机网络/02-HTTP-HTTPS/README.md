## 📚 核心内容

### 1. HTTP 协议
- HTTP 请求/响应模型
- HTTP 方法（GET、POST、PUT、DELETE 等）
- HTTP 状态码

- 安全配置

### 4. HTTP 缓存

- Cache-Control 头
- ETag 与 Last-Modified
- 缓存策略
- CDN 缓存
