# Warp 快捷键

> Warp 是一款现代化的终端工具，提供了丰富的快捷键来提升开发效率。

## 窗口与标签管理

| 快捷键            | 功能         |
| ----------------- | ------------ |
| `Cmd + W`         | 关闭当前标签页 |
| `Cmd + T`         | 新建标签页     |
| `Cmd + N`         | 新建窗口       |
| `Cmd + D`         | 分屏（竖分）   |
| `Cmd + Shift + D` | 分屏（横分）   |

## 屏幕与行操作

| 快捷键      | 功能                 |
| ----------- | -------------------- |
| `Cmd + K`   | 清空屏幕             |
| `Cmd + L`   | 清空当前行           |
| `Cmd + A`   | 选择当前行的所有文本 |

## 光标移动

| 快捷键   | 功能         |
| -------- | ------------ |
| `Ctrl + A` | 光标移到行首 |
| `Ctrl + E` | 光标移到行尾 |

## 其他功能

| 快捷键    | 功能         |
| --------- | ------------ |
| `Cmd + ,` | 打开设置     |
| `Cmd + /` | 打开命令面板 |
| `Ctrl + R` | 搜索历史命令 |
| `Tab`     | 自动补全     |
