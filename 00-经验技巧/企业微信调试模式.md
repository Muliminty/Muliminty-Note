# 企业微信调试模式

> 介绍如何在不同系统下开启企业微信的调试模式，使用开发者工具进行问题排查

---

## 开启调试模式

### Windows 系统

**快捷键**：`Ctrl + Alt + Shift + D`

1. 按下快捷键 `Ctrl + Alt + Shift + D` 进入调试模式
2. 在调试菜单中，右键选择 `showDevTools` 打开开发者工具

### Mac 系统

**快捷键**：`Ctrl + Command + Shift + D`

1. 按下快捷键 `Ctrl + Command + Shift + D` 开启调试模式
2. 在左上角菜单栏选择 **调试** → **开启 webView 审查元素**
3. 在需要调试的页面上点击右键，选择 **检查元素**，即可打开开发者工具

## 使用开发者工具

打开开发者工具后，你可以：

- 查看控制台（Console）：查看日志、错误信息
- 查看网络（Network）：监控网络请求
- 查看源码（Sources）：调试 JavaScript 代码
- 查看元素（Elements）：检查 DOM 结构
- 进行各种调试操作

## 注意事项

1. 调试模式仅在开发环境使用，生产环境应关闭
2. 某些企业微信版本可能快捷键略有不同
3. 如果快捷键无效，可以尝试重启企业微信客户端
