# 总览与索引目录说明

本目录是整个知识库的全局导航层。

职责：

- 提供全库总入口
- 汇总一级主题入口
- 提供迁移期的过渡导航

不承担职责：

- 不承载具体知识正文
- 不承载归档资料

维护规则：

- 全局导航统一写入 `!MOC-总览与索引.md`
- 若后续新增总览页面，应放在本目录下并保持高层次导航属性
